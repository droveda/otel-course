# OpenTelemetry SpringBoot Starter
* Simplifies integration of OpenTelemetry with Spring Boot applications.
* Provides auto-configuration for tracing, metrics, and logs.
* Reduces boilerplate setup by wiring exporters, propagators, and samplers automatically.
* Enables developers to observer applications with minimal configuration changes.
* Developed and managed by OpenTelemetry Community.
* Automatic Instrumentation
* Manual Instrumentation
  * We can autowire OpenTelemetry and create custom spans, metrics, etc...
